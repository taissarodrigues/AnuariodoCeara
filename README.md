## Características

- Página de mapas com conteúdo organizado em accordion
- Layout responsivo
- Integração com Bootstrap para componentes interativos
- Estilização customizada seguindo identidade visual do projeto
- Header reutilizável com navegação
- Redirecionamento ao clicar no ícone de busca (lupa)

## Comportamento da Aplicação

- O accordion permite expandir e recolher conteúdos de mapas
- A navegação entre páginas é feita de forma fluida
- O clique no ícone de busca redireciona para a página de pesquisa
- A interface se adapta a diferentes tamanhos de tela
- Componentes visuais seguem padronização via variáveis CSS
